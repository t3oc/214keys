# 214 ключей · 康熙部首

Интерактивный справочник по всем **214 радикалам Канси** — с японскими и китайскими чтениями, озвучкой и объяснениями на русском.

**Демо:** [laditeo.github.io/214keys](https://laditeo.github.io/214keys/)

---

## Возможности

- **Таблица радикалов** — все 214 ключей, сгруппированных по числу черт
- **Поиск** — по иероглифу, чтению, pinyin и тексту объяснения
- **Карточка ключа** — японское и китайское чтение, варианты написания, число черт, пояснение на русском
- **Озвучка** — кнопки 🔊 у чтений; аудио подгружается по группам черт
- **Клик по большому иероглифу** — тематический звуковой эффект (по emoji радикала) + анимация «салют»; **Пробел** — то же в модалке
- **Мини-карта** — в модальном окне слева: вся таблица целиком, рамка видимой области, drag/swipe для прокрутки, клик по кандзи; кнопка сворачивания снаружи панели
- **Навигация** — ←/→ между ключами, ↑/↓ между группами по чертам; те же переходы — кнопками над и под карточкой
- **Тема** — светлая / тёмная (сохраняется в `localStorage`)
- **Шрифт иероглифов** — классический serif или монолинейный sans

---

## Как пользоваться

1. Откройте сайт и найдите нужный ключ в таблице или через поиск.
2. Кликните по иероглифу — откроется карточка с чтениями и объяснением.
3. Нажмите на динамик рядом с **日本語** или **中文**, чтобы услышать произношение.
4. В модалке:
   - **← / →** — предыдущий / следующий ключ в текущей выборке
   - **↑ / ↓** — предыдущая / следующая группа по числу черт
   - **Esc** — закрыть карточку
   - **Мини-карта слева** — клик по ячейке переключает ключ; drag рамки или свайп по карте прокручивает фон; tap вне рамки — прыжок к месту в таблице
5. Переключатели **Светлая / Тёмная** и **Классика / Монолиния** — над полем поиска.

> Мини-карта по умолчанию открыта в альбомной ориентации и свёрнута в портретной.

## Локальный запуск

Сайт — статический, без сборки. ES-модули требуют локальный сервер (не `file://`).

```bash
# Python
python -m http.server 8080

# или Node
npx serve .
```

Откройте `http://localhost:8080`.

## Обновление данных

1. Отредактируйте кортежи в `build_data.py`.
2. Перегенерируйте JSON:

```bash
node gen_json.js
```

Скрипт обновит `js/radicals.js` и `data/radicals.json`.

## Генерация аудио

Озвучка создаётся через Microsoft Edge TTS (`msedge-tts`):

```bash
npm install
npm run generate:audio
```

Файлы пишутся в `audio/jp/` и `audio/cn/` (по одному MP3 на радикал). Лог генерации — в `.gitignore`.

### Звуковые эффекты (клик по иероглифу)

214 коротких `.ogg` подбираются по emoji радикала из [Kenney's Sound Pack](https://kenney.nl/assets) (CC0). Скрипт скачивает их с зеркала gamesounds.xyz:

```bash
npm run generate:sfx
```

Файлы — в `audio/sfx/001.ogg` … `214.ogg`. Если файл недоступен, подставляется запасной UI-звук; в браузере при ошибке загрузки остаётся синтетический «pop».
